# 智能家居手势控制平台 v2.0 (Smart Home Gesture Control Platform)

本项目是一个基于 **ESP32-S3** 主控芯片开发的非接触式智能家居控制终端原型 。通过集成红外手势识别、RGB 氛围灯光调节、OLED 状态显示及蜂鸣器音效反馈，实现了全方位的空间手势交互操控 。  

## ─ 主档特性 ─

- **非接触式手势交互**：利用 `PAJ7620U2` 传感器精准捕捉并识别 6 种空间动作（上挥、下挥、左挥、右挥、顺时针旋转、前推） 。  
- **多场景灯光模式**：内置 Warm（温馨）、Cinema（影院）、Reading（阅读）、Romantic（浪漫）4 种预设 RGB 色温 。  
- **多模态状态反馈**：
  - **视觉**：通过 0.96 寸 OLED 屏实时显示开关状态、当前模式、亮度进度条 。  
  - **听觉**：无源蜂鸣器针对不同手势输出差异化音效，切换模式时具备阶梯式音调反馈 。  
- **能耗估算**：边缘端根据当前灯光亮度和开启状态，动态进行简单的线性实时功率（W）估算并直观显示 。  

## ─ 硬件架构与引脚连接 ─

| **模块名称**              | **接口类型**         | **ESP32-S3 引脚**                 | **备注说明**            |
| ------------------------- | -------------------- | --------------------------------- | ----------------------- |
| **WS2812B RGB灯带**       | 单总线 (Single Line) | `GPIO 13`                         | 驱动 16 颗串联 RGB 灯珠 |
| **无源蜂鸣器**            | 信号控制 (PWM/IO)    | `GPIO 15`                         | 状态音效提示            |
| **I2C 总线设备**          | I2C                  | `SDA -> GPIO 12` `SCL -> GPIO 11` | 传感器与 OLED 共用总线  |
| **SSD1306 OLED (128x64)** | I2C                  | 同上（默认地址：`0x3C`）          | 界面信息反馈显示        |
| **PAJ7620U2 手势传感器**  | I2C                  | 同上                              | 空间动作捕捉            |

## ─ 操控指南 ─

系统上电初始化成功后会发出提示音 ，此时可通过以下手势“隔空”控制系统：  

- **右挥手 (`GES_RIGHT_FLAG`)**：开启灯光 。  
- **左挥手 (`GES_LEFT_FLAG`)**：关闭灯光 。  
- **前推动作 (`GES_FORWARD_FLAG`)**：反转当前灯光状态（开启时关闭，关闭时开启） 。  
- **上挥手 (`GES_UP_FLAG`)**：增加灯光亮度（每次步进 51，最高至 255） 。  
- **下挥手 (`GES_DOWN_FLAG`)**：降低灯光亮度（每次步进 51，最低至 0） 。  
- **顺时针旋转 (`GES_CLOCKWISE_FLAG`)**：循环切换 4 种灯光场景模式（温馨 $\rightarrow$ 影院 $\rightarrow$ 阅读 $\rightarrow$ 浪漫），并伴随递增音调 。  

## ─ 依赖的软件库 ─

在编译本工程前，请确保在 Arduino IDE 的库管理器中安装了以下开源依赖库：

1. `Adafruit_NeoPixel`（用于驱动 WS2812B 灯带）   
2. `Adafruit_SSD1306` & `Adafruit_GFX`（用于 OLED 屏幕驱动与图形渲染）   
3. `paj7620`（手势传感器官方驱动库）   

## ─ 调试日志 ─

代码中集成了串口日志输出，通信波特率为 **`115200`** 。当触发手势时，可通过串口监视器查看实时运行状态：  

Plaintext

```

Gesture:2  Light:ON   Mode:Cinema    Bright:128
Gesture:3  Light:OFF  Mode:Romantic  Bright:128
```
